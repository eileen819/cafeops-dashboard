# CafeOps Dashboard 화면 및 라우트 구조

## 1. 라우트 목록

| 경로       | 화면          | 목적                                   | 우선순위 |
| ---------- | ------------- | -------------------------------------- | -------- |
| /          | DashboardPage | 매장 운영 요약 확인                    | 필수     |
| /inventory | InventoryPage | 재고 목록, 검색, 필터, 정렬, 등록/수정 | 필수     |
| /sales     | SalesPage     | 판매 흐름 및 차트 확인                 | 선택     |
| /settings  | SettingsPage  | 운영 기준 설정                         | 제외     |

---

## 2. DashboardPage

### 목적

운영자가 오늘의 매출, 주문 수, 재고 부족 품목, 발주 필요 품목을 한눈에 확인할 수 있도록 한다.

### 포함 영역

| 영역                 | 설명                                                             |
| -------------------- | ---------------------------------------------------------------- |
| Summary Cards        | 오늘 매출, 이번 주 주문 수, 재고 부족 품목 수, 발주 필요 품목 수 |
| Low Stock List       | 부족 상태 품목 목록                                              |
| Sales Trend Chart    | 일자별 매출 추이                                                 |
| Category Sales Chart | 카테고리별 판매 비중                                             |

### 사용자 행동

| 행동                | 결과                                   |
| ------------------- | -------------------------------------- |
| 재고 부족 품목 클릭 | /inventory로 이동하거나 해당 품목 강조 |
| 차트 확인           | 판매 흐름 파악                         |
| 재고 관리 버튼 클릭 | /inventory로 이동                      |

---

## 3. InventoryPage

### 목적

운영자가 전체 재고 품목을 확인하고, 검색/필터/정렬을 통해 발주 필요 품목을 빠르게 찾을 수 있도록 한다.

### 포함 영역

| 영역                       | 설명                         |
| -------------------------- | ---------------------------- |
| Inventory Filter Bar       | 검색, 카테고리 필터, 정렬    |
| Inventory Table            | 재고 목록 표시               |
| Inventory Status Badge     | 부족 / 정상 / 과잉 상태 표시 |
| Recommended Order Quantity | 권장 발주 수량 표시          |
| Upsert Inventory Modal     | 상품 등록 / 수정 폼          |

### 사용자 행동

| 행동           | 결과                       |
| -------------- | -------------------------- |
| 검색어 입력    | 상품명 기준 필터링         |
| 카테고리 선택  | 해당 카테고리 품목만 표시  |
| 정렬 선택      | 선택 기준으로 목록 정렬    |
| 상품 등록 클릭 | 등록 모달 열림             |
| 상품 수정 클릭 | 수정 모달 열림             |
| 폼 제출        | mock API 요청 후 목록 갱신 |
